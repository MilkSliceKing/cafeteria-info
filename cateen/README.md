# 食堂信息查询系统

这是一个静态网页应用，用于展示食堂摊位、菜品和用户评价信息。

## 功能特点

- 摊位信息展示（位置、营业时间、描述）
- 菜品浏览（价格、描述、图片）
- 用户评价查看（评分、评论）
- 响应式设计，适配各种设备
- 简洁美观的界面

## 使用方法

1. 打开 `index.html` 文件即可使用
2. 点击顶部导航按钮切换不同内容区域
3. 点击摊位卡片可查看详细信息

## 数据文件说明

- `data/stalls.json` - 摊位信息
- `data/dishes.json` - 菜品信息
- `data/reviews.json` - 用户评价

## 如何更新数据

1. 修改对应的JSON文件
2. 保存文件
3. 刷新网页即可看到更新后的内容

## 注意事项

- 图片路径需要根据实际情况调整
- 数据格式请保持JSON格式正确
- 建议使用文本编辑器修改数据文件

## 技术栈

- HTML5 + CSS3 + JavaScript
- 响应式设计
- 本地数据存储

## 部署方式

可以直接在本地打开使用，也可以部署到任何静态网站托管服务（如GitHub Pages、Netlify等）。
